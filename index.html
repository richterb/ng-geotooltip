What is Angular-GeoTooltip?

It is a simple AngularJS directive that you can use in your content to display geographical tooltips when hovering your text. It can be a 2D point (latitude & longitude), a GeoJSON shape, or both. It is designed to be smart about its positioning, so that you don't have to fight against vicious scrollbars. It is also highly configurable, both using a global configuration, or using attributes on each directive. Angular-GeoTooltip is based on the wonderful LeafletJS library to display tile-based maps, and can be configured to use any tile provider such as MapBox; by default it will use the free MapQuest OSM Tiles.

<geotooltip geojson="geoJson" coords="pointCoordinates" delay="0" width="600" height="400">Hello, I am a tooltip!</geotooltip>
